# Siddharth Taurani — Portfolio

> **Live:** [your-domain.com](https://your-domain.com)

Personal portfolio website showcasing my projects, experience, and skills in AI/ML engineering.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** for bundling
- **Tailwind CSS** for styling
- **Framer Motion** for animations
- **shadcn/ui** for base components

## Getting Started

```bash
npm install
npm run dev
```

The dev server runs at [http://localhost:8080](http://localhost:8080).

## Build

```bash
npm run build
npm run preview
```
